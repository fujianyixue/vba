对象名 含义
ActiveWorkbook 当前工作簿
ActiveSheet 当前工作表
ActiveCell 当前工作表活动单元格
ActiveChart 当前工作簿活动图表
Selection 被选中的部分
Application 应用程序的工作环境
Workbooks 当前所有打开的工作簿对象的集合
Workbook 代表一个独立的工作簿文件
Worksheets 代表当前工作簿所有的工作表的集合
Worksheet 代表工作簿一个独立的工作表
Range 代表一个半桥区域，可以由多个半桥组成，也可是由一个单元格组成
Application
属性 说明
Caption 应用程序标题栏显示的文本
Cursor 鼠标的形状
DisplayAlerts 系统的应用提示和警告信息
DisplayFormulaBar 用以确定是否显示公式编辑栏
DisplayScrollBars 用以确定是否显示Excel的滚动条
DisplayStatusBar 用以确定是否显示Excel的状态栏
EnableCancelkey 用以确定是否允许用户中断正在执行着的VBA程序
ScreenUpdating 用以确定是否显示VBA程序的中间运算结果
StatusBar 设置状态栏的文本
ActiveWorkbook
ActiveSheet
ActiveCell
ThisWorkBook 用以返回正在执行着VBA程序所在的工作簿对象
PathSeparator 返回文件文件路径分隔符“\”，该属性是一个只读属性
UserName 设置当前用户名称
方法 说明
Calculate 对Excel打开的工作簿中所有的公式进行重新计算
Goto 选择工作簿中的一个Range对象或一个VBA过程
InputBox 用于打开一个输入对话框，允许用户输入数据
Onkey 按下某个或组合键时执行一段VBA代码
Workbooks
属性 说明
Count 只读，返回Excel打开的工作簿数
方法 说明
Add 向工作簿对象集中一个新的工作簿
Open 打开一个指定的工作簿
Close 关闭一个工作簿文件
Workbook
属性 说明
ActiveSheet 返回当前工作簿中的活动工作表对象
Author 返回或设定工作簿的作者姓名
Saved 用于瓢工作簿文件是否做过修改
方法 说明
Activate 用于使一个工作簿为当前活动工作簿
Close 用于关闭指定的工作簿
Save 用于保存指定的工作簿
SaveAs 用于另存为指定的工作簿
Worksheets
属性 说明
Count 返回当前工作簿中的的工作表的数量
方法 说明
Add 向工作表对象集中添加一个新的工作表对象
Worksheet
属性 说明
Name 设置或返回工作表对象的名称
Visible 确定是否隐藏某个工作表
方法 说明
Calculate 对指定的工作表中的所有公式进行重新计算
Copy 用于复制指定的工作表
Move 移动指定的工作表
Delete 无参数删除指定的工作表
Cell 获取指定工作表指定行列的某个单元格
Range
属性 说明
FormulaR1C1 为指定的单元格或单元格区域建立和存储公式
Value 为指定的单元格或单元格区域赋值
Cell 以指定的单元格为苦战来描述被引用的单元格
Offset 以指定的单元格为苦战来描述被引用的单元格
Name 返回或设定指定的单元格或单元格区域的名称
Count 返回指定的单元格区域中包含的单元格的数目
CurrentRegion 返回以某个单元格为基点的存储有数据的一个连结的单元格区域
方法 说明
Select 用于选择指定的单元格区域
Activate 用于选择指定的单元格并指定其成为活动单元格
AutoFit 调整选定的单元格区域所在行和列的行高与列宽，使其行高与列宽自动设定的最佳的高度与宽度
Clear 清除指定的单元格区域的内容，格式和批注
ClearContents 清除指定的单元格区域的内容
ClearFormats 清除指定的单元格区域的格式
ClearComments 清除指定的单元格区域的批注
ClearNotes 清除指定的单元格区域的批注
Copy 将指定单元格区域中的数据库复制到剪贴板或指定的目的单元格区域中
Cut 将指定单元格区域中的数据库剪切到剪贴板或指定的目的单元格区域中
PasteSpecial 用于将剪贴板中复制或剪切下来的单元格区域中的数据库有选择地粘贴到指定的目的单元格区域中
